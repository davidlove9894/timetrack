Firebase sync setup

  1. Go to console.firebase.google.com
  2. Create a free project → Add a Web app
  3. Copy the firebaseConfig object shown
  4. Open timetrack.html in a text editor
  5. Replace FIREBASE_CONFIG values at the top
  6. In Firebase: Firestore Database → Create → Start in test mode
  7. Host on Vercel: vercel.com → drag & drop the file → done
  8. Share the URL with your EA
